# Yolov4_RDD_detection  
## Usage
`git clone https://github.com/Roger-Lo2000/yolov4_RDD_detection.git`  
  
`bash GCP_init.sh`  
  
`bash init.sh`  
  
Restart VM  
  
`conda create -n {name} python=3.6`  
  
`conda activate {name}`  
  
`cd yolov4_RDD_detection.git`  
  
`pip install -r requirements`
  
`bash all.sh`
